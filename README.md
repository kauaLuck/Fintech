## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->
O Fintech é um projeto desenvolvido durante meus estudos em Front-end com o objetivo de praticar a criação de páginas web utilizando HTML e CSS. Durante o desenvolvimento, foram aplicados conceitos de semântica HTML, responsividade, organização de código e boas práticas de estruturação de interfaces, resultando em uma landing page moderna e bem estruturada.
